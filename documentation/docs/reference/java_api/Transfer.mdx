

### builder(address, amount, output_kind): [TransferBuilder](#transferbuilder)

Creates a new Builder for making a Transfer object

| Parameter       | Type |  Description |
| --------------- | -------- | -------- |
| address | [AddressWrapper](#addresswrapper) |  The addres we want to send to |
| amount | long |  The amount we want to send |
| output_kind | [OutputKind](#outputkind) |  The kind of output we wish to use, can be OutputKind.NONE |
