### messages(messages): [AccountInitialiser](#accountinitialiser)

Messages associated with the seed.
 The account can be initialised with locally stored messages.

| Parameter       | Type |  Description |
| --------------- | -------- | -------- |
| messages | [Message[]](#message) |   |

### addresses(addresses): [AccountInitialiser](#accountinitialiser)

Address history associated with the seed.

| Parameter       | Type |  Description |
| --------------- | -------- | -------- |
| addresses | [Address[]](#address) |   |

### skipPersistence(): [AccountInitialiser](#accountinitialiser)

Skips storing the account to the database.

### allowCreateMultipleEmptyAccounts(): [AccountInitialiser](#accountinitialiser)

Enables creating multiple accounts without history.
 The wallet disables it by default to simplify account discovery.

### initialise(): [Account](#account)

Initialises the account.
