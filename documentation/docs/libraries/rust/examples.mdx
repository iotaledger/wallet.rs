You can see the examples in the library's [examples directory](https://github.com/iotaledger/wallet.rs/tree/develop/examples).
You can list all available examples by running the following command:

```bash
cargo run --example # lists the available examples
```

To run an example, you can use the following command, replacing _01_create_wallet_ with the desired example:

```bash
cargo run --example 01_create_wallet # execute the `01_create_wallet` example
```
