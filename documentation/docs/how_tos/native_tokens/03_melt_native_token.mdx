You may want to melt some of your circulating native tokens. To do so, you will need to call the
`Account.melt_native_token(native_token, options)` function.

If you melt your native tokens, you can still destroy the foundry that created them.

2. Get Alice's account which was [created in the first guide](../accounts_and_addresses/01_create_a_wallet_account.mdx).
3. Get the account's balance.
4. Melt 10 native tokens of the supplied ID.
