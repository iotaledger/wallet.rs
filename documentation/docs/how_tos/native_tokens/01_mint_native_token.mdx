You can use a node running the stardust update to mint
[native tokens](https://wiki.iota.org/shimmer/introduction/explanations/what_is_stardust/tokenization#native-tokens).
To do so, you will only need to [add some funds to your account](../accounts_and_addresses/03_request_funds.mdx), define
the native token's options and call the `Account.mint_native_token(native_token_options, options)` function.

<TipFaucet/>

## Code Example

The following example will:

1. Create an account manager.
2. Get Alice's which was [created in the first guide](../accounts_and_addresses/01_create_a_wallet_account.mdx).
3. Create the `NativeTokenOptions`.
4. Mint the native token by calling the `Account.mint_native_token(native_token_options, options)` function.
