You may want to burn some of your circulating native tokens. To do so, you will need to call the
`Account.burn_native_token(native_token, options)` function.

:::warning

If you burn your native tokens, you will not be able to destroy the foundry that created them. If you want to destroy
the foundry in the future, you should consider [melting your native tokens](03_melt_native_token.mdx)

:::
