You can destroy an alias output by ID using the `Account.destroy_alias(alias_id, options)` function. Any outputs
controlled by the alias will be swept unless they have a
[storage deposit return](https://wiki.iota-community.org/shimmer/introduction/develop/explanations/ledger/simple_transfers#storage-deposit-returns-in-transfers),
[time-lock](https://wiki.iota-community.org/shimmer/introduction/develop/explanations/ledger/simple_transfers#timelocking-funds-in-transfers)
or [expiration unlock](https://wiki.iota-community.org/shimmer/introduction/develop/explanations/ledger/simple_transfers#transfer-of-funds-with-expiration)
conditions.

## Code Example

The following example will:

1. Create an account manager.
2. Get Alice's account which was [created in the first guide](../accounts_and_addresses/01_create_a_wallet_account.mdx).
3. Get the account's balance.
4. Destroy an [alias output](https://wiki.iota-community.org/shimmer/introduction/develop/explanations/ledger/alias) by id.
5. Get the account's balance again to show the difference after step 4.
