You can destroy an alias output by ID using the `Account.destroy_alias(alias_id, options)` function. If the alias still owns any outputs when you try to destroy it, you will get an error.   

## Code Example

The following example will:

1. Create an account manager.
2. Get Alice's account which was [created in the first guide](../accounts_and_addresses/01_create_a_wallet_account.mdx).
3. [Get the account's balance](./../accounts_and_addresses/04_check_balance.mdx).
4. Destroy an [alias output](https://wiki.iota.org/shimmer/introduction/explanations/ledger/alias) by id.
5. Get the account's balance again to show the difference after step 4.
